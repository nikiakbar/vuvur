Application Context: Vuvur (Android Client)
- Application Overview
Name: Vuvur
Type: A native Android media gallery client.
Purpose: To browse, stream, and manage a remote library of images and videos served by a custom backend API.
Architecture: MVVM (Model-View-ViewModel) using modern Android Jetpack libraries.
Language: Kotlin.
- Key Technologies & Stack
UI Framework: Jetpack Compose (Material 3).
Navigation: Jetpack Navigation Compose.
Networking: Retrofit 2 & OkHttp 4.
Image Loading: Coil (Coroutines Image Loader), configured with a custom OkHttpClient for authentication.
Video Playback: Media3 ExoPlayer.
Local Storage: Jetpack DataStore Preferences (for settings like API URL and Zoom level).
Concurrency: Kotlin Coroutines & Flows.
- Core Functionality & Features
Gallery Browsing:
- Displays a staggered grid of media thumbnails using LazyVerticalStaggeredGrid.
- Supports pagination (infinite scrolling) triggered when reaching the end of the list.
- Filtering: Filters by Tags (search query), Groups, and Subgroups.
- Sorting: Supports sorting by Random, Oldest First, and Newest First.
- Pull-to-Refresh: Implements Material 3 PullToRefreshBox to reload data.
Media Viewing:
- Viewer: A full-screen VerticalPager that allows swiping through media items.
- Images: Supports deep zooming and panning using detectTransformGestures and detectTapGestures.
- Video: Streams video using ExoPlayer with PlayerView.
Management:
- Deletion: Allows deleting media items, which moves them to a recycle bin on the server.
Settings:
- Allows switching between multiple API endpoints (Dev/Prod).
- Allows adjusting the default double-tap zoom level.
- Triggers backend cache cleanup.
- Critical Implementation Details
Authentication (API Key Injection):
- The backend requires an X-Api-Key header.
- Retrofit: A custom OkHttp interceptor injects this header into all API calls.
- Coil (Images): A custom ImageLoaderFactory is implemented in VuvurApplication to inject the header into image requests.
- ExoPlayer (Video): A DefaultHttpDataSource.Factory is used to inject the header via setDefaultRequestProperties.
Video Playback Control:
- To prevent videos from playing automatically when pre-loaded by VerticalPager, the MediaSlide component accepts an isCurrentlyVisible boolean.
- A LaunchedEffect monitors this boolean and sets exoPlayer.playWhenReady accordingly.
API Endpoints:
- Base URL is configurable.
- Key endpoints: /api/gallery, /api/stream/{id}, /api/thumbnails/{id}, /api/gallery/groups.
- Data Structures
- MediaFile: Represents a file with fields: id, path, type (image or video), width, height, mod_time, exif.
- GalleryUiState: A sealed interface managing UI states (Loading, Scanning, Error, Success). The Success state holds the list of files, pagination status, current API configuration, and active filters.
